# Sand Saga

React-based web app with Material UI, and TailwindCSS. Dockerized for easy deployment.

## Tech Stack
- React 19
- Material UI
- GSAP
- TailwindCSS
- Docker & Docker Compose
- React Testing Library

## Setup

### Prerequisites
- Node.js v18+
- Docker & Docker Compose

### Install & Run
```bash
git clone https://github.com/your-username/sand-saga.git
cd sand-saga
npm install
npm start
